## Overview

The Agent Control Dashboard is a Next.js + React web application for managing your agents and controls through a visual interface.

## Running the Dashboard

```bash
cd ui
pnpm install
pnpm dev
```

Dashboard runs at `http://localhost:4000`.

## Features

- **Agent management** — View registered agents and their policies
- **Control editor** — Create and edit controls with a visual form
- **Policy builder** — Assemble control sets into policies
- **Observability** — View control execution logs and stats

## Configuration

| Variable | Default | Description |
|----------|---------|-------------|
| `NEXT_PUBLIC_API_URL` | `http://localhost:8000` | Backend API URL |
