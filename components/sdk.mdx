## Overview

The Python SDK (`agent-control-sdk`) provides decorator-based protection for your agent functions.

## Installation

```bash
pip install agent-control-sdk
```

## Quick Usage

```python
import agent_control
from agent_control import control, ControlViolationError

# Initialize
agent_control.init(
    agent_name="my-agent",
    agent_id="support-agent-v1",
    server_url="http://localhost:8000"
)

# Protect any function
@control()
async def chat(message: str) -> str:
    return await llm.generate(message)

# Handle violations
try:
    response = await chat("Hello!")
except ControlViolationError as e:
    print(f"Blocked: {e.message}")
```
